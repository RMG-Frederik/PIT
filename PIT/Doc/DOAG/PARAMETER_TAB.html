VALIDATION_STRING VARCHAR2(4000) Y   Validation chunk, PL/SQL fragment. Must provide a boolean value, without semikolon. References parameter values via #STRING#|#DATE#|#FLOAT#|#INTEGER#
VALIDATION_MESSAGE VARCHAR2(4000) Y   Message to log if parameter validation was unsuccesful
RELEASE_VERSION VARCHAR2(40) Y   Release of first deployment of this parameter. Used to check, whether all parameters were succesfully delivered upon last deployment.
