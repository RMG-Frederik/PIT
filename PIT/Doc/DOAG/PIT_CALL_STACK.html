Table PIT_CALL_STACK

table for trace and time information
Columns
Name Type Optional Default Comments
ID NUMBER     Primary key, deliverd with the log message.
CALL_LEVEL NUMBER Y   Increment for each step of the call stack
ACTION VARCHAR2(5) Y   Activity of the current step: method name, line-number or user defined action
SESSION_ID VARCHAR2(256) Y   Session-Id, required to attach log messages to sessions
USER_NAME VARCHAR2(30) Y   connected database user
METHOD_NAME VARCHAR2(30) Y   method name of the package
MODULE_NAME VARCHAR2(192) Y   package name
WALL_CLOCK TIMESTAMP(6) Y   timestamp upon creation of the trace message
ELAPSED NUMBER Y   time consumed by the procedure (w/o time spent in called procedures)
ELAPSED_CPU NUMBER Y   CPU-time consumed by the procedure (w/o time spent in called procedures)
TOTAL_TIME INTERVAL DAY(2) TO SECOND(6) Y   time consumed by the procedure (including time spent in called procedures)
TOTAL_TIME_CPU NUMBER Y   CPU-time consumed by the procedure (including time spent in called procedures)
