edit_parameter_group 

Procedure edit_parameter_group(p_parameter_group_id varchar2,
                               p_group_description  varchar2,
                               p_is_modifiable      varchar2)

Procedure to change the definition of a parameter group

Usage
Is called by the developer to maintain the settings of a parameter group. May be used to create a new parameter group as well.

Parameters
p_parameter_group_id   Name of the parameter group
p_group_description   Longtext description of the parameter group
p_is_modifiable   Flag indicating whether parameters within this group are allowed to be modifiable by the end user

edit_parameter_type 

Procedure edit_parameter_type(p_parameter_type_id varchar2,
                              p_type_description  varchar2)

Procedure to edit the definition of a parameter type.

Usage
Procedure is called by the developer to edit parameter types. Paramater types are used to define the kind of string type a parameter has. Examples are HTML, SQL or similar. They are used to control which editer is used to edit the parameter value on the UI

Parameters
p_parameter_type_id   Name of the parameter type
p_type_description   Longtext description of the parameter type

edit_parameter 

Procedure edit_parameter(p_parameter_id          in varchar2,
                         p_parameter_group_id    in varchar2,
                         p_parameter_description in varchar2,
                         p_string_value          in clob default null,
                         p_xml_value             in xmltype default null,
                         p_integer_value         in number default null,
                         p_float_value           in number default null,
                         p_date_value            in date default null,
                         p_timestamp_value       in timestamp with time zone default null,
                         p_boolean_value         in varchar2 default null,
                         p_is_modifiable         in varchar2 default null,
                         p_parameter_type_id     in varchar2 default null,
                         p_validation_string     in varchar2 default null,
                         p_validation_message    in varchar2 default null)

Procedure to edit a parameter

Usage
Is called from the developer to create or change a parameter.

Parameters
p_parameter_id   Name of the parameter
p_parametergroup   Name of the parameter group
p_string_value   CLOB parameter value
p_xml_value   XML parameter value
p_integer_value   Integer parameter value
p_float_value   Float parameter value
p_date_value   Date parameter value
p_timestamp_value   Timestamp with time zone parameter value
p_boolean_value   Boolean parameter value
p_is_modifiable   Flag indicating whether this parameter is modifiable by the end user
p_parameter_type   String type of the parameter, Reference to PARAMETER_TYPE.
p_validation_string   PL/SQL-chunk to validate a parameter value entered. Is used to validate parameter values
p_validation_message   Name of a Message that is thrown if a validation of the parameter fails. May be any predefined message from package MSG
