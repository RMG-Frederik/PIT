Table MESSAGE

central log message repository in various languages.
Columns
Name Type Optional Default Comments
MESSAGE_NAME VARCHAR2(120)     Unique message name. Used to reference the message as a constant in package MSG.
MESSAGE_LANGUAGE VARCHAR2(120)     Languae reference of the message. Taken from Oracle v$nls_valid_values
MESSAGE_TEXT CLOB Y   message text. Positional replacement with pattern #n#
SEVERITY NUMBER(3) Y   severity of the message. Allowed values between log.level_fatal and log.level_all
CUSTOM_ERROR_NUMBER NUMBER Y   Optional pointer to an oracle error number. If null, an exception will be generated. To define custom errors, use null.
